Навыки, умения и сертификаты

Приветствую Вас на своем сайте-портфолио.

Меня зовут Галина. Живу в Санкт-Петербурге. У меня высшее техническое образование и большой жизненный опыт. Сейчас я занимаюсь любимым делом - программированием. Создаю сайты и веб-приложения на основе HTML5/CSS3 и JavaScript. Освоила JS-библиотеки React и Redux для создания пользовательских интерфейсов. Продолжаю обучение - на очереди изучение Node.

Владею навыками работы с фреймворком Bootstrap5. Использую кросс-платформенный онлайн-сервис Figma и библиотеку анимации GSAP.

Выполняю адаптивную верстку. Благодаря адаптивной верстке сайт будет качественно и удобно отображаться на разных устройствах, таких как персональные компьютеры, ноутбуки, планшеты, смартфоны...

Мои инструменты и сертификаты

HTML (Hypertext Markup Language) - это код, который используется для структурирования и отображения веб-страницы и её контента.
Сертификат канадской школы "CanSheCode inc" и образовательной онлайн-платформы Stepik ⇒

CSS — Cascading Style Sheets — это каскадные таблицы стилей, которые отвечают за описание внешнего вида веб-сайта и делают его более привлекательным для читателя.
Сертификат канадской школы "CanSheCode inc" и образовательной онлайн-платформы Stepik
⇐

JavaScript— это лучший друг HTML и CSS. Этот язык оживляет сайты. С помощью кода на JavaScript программист определяет, как страница отреагирует на действия пользователя. ⇒

Bootstrap — это открытый HTML, CSS и JS фреймворк, который используется веб-разработчиками для быстрой вёрстки адаптивных дизайнов сайтов и веб-приложений. Сертификат канадской школы "CanSheCode inc"
⇐

Figma — кросс-платформенный онлайн-сервис для дизайнеров интерфейсов и веб-разработчиков. Сертификат канадской школы "CanSheCode inc" ⇒
